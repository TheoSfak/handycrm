2. ΓΡΗΓΟΡΗ ΕΓΚΑΤΑΣΤΑΣΗ (ΣΥΝΙΣΤΩΜΕΝΟ) 🚀
══════════════════════════════════════════════════════════════════════════

ΒΗΜΑ 1: Αποσυμπίεση
   - Αποσυμπιέστε όλα τα αρχεία στον φάκελο του web server σας
   - Για XAMPP: C:\xampp\htdocs\handycrm\
   - Για άλλους servers: public_html/handycrm/ ή www/handycrm/

ΒΗΜΑ 2: Ανοίξτε το install.php
   Πηγαίνετε στον browser:
   
   http://localhost/handycrm/install.php
   
   ή
   
   http://your-domain.com/install.php

ΒΗΜΑ 3: Συμπληρώστε τις ρυθμίσεις
   
   📝 Database Host: localhost
   📝 Database Port: 3306
   📝 Database Name: handycrm (θα δημιουργηθεί αυτόματα αν δεν υπάρχει)
   📝 Username: root (ή το username της MySQL σας)
   📝 Password: (αφήστε κενό για XAMPP, ή το password σας)

ΒΗΜΑ 4: Πατήστε "Εγκατάσταση"
   
   Το σύστημα θα:
   ✓ Δημιουργήσει τη βάση δεδομένων αυτόματα
   ✓ Εισάγει όλους τους πίνακες
   ✓ Εισάγει τα δεδομένα (χρήστες, παραδείγματα)
   ✓ Δημιουργήσει το αρχείο config/config.php

ΒΗΜΑ 5: Διαγραφή install.php
   
   ⚠️ ΣΗΜΑΝΤΙΚΟ: Μετά την επιτυχημένη εγκατάσταση, πατήστε το κουμπί
   "Διαγραφή install.php" για λόγους ασφαλείας!

ΒΗΜΑ 6: Σύνδεση
   
   Πηγαίνετε στο: http://localhost/handycrm/
   
   Χρησιμοποιήστε έναν από τους λογαριασμούς παρακάτω.

4. ΧΕΙΡΟΚΙΝΗΤΗ ΕΓΚΑΤΑΣΤΑΣΗ (Προχωρημένοι)
══════════════════════════════════════════════════════════════════════════

Αν προτιμάτε χειροκίνητη εγκατάσταση:

1. Δημιουργήστε τη βάση δεδομένων:
   
   CREATE DATABASE handycrm CHARACTER SET utf8mb4 COLLATE utf8mb4_unicode_ci;

2. Εισάγετε το SQL αρχείο:
   
   Via phpMyAdmin:
   - Επιλέξτε τη βάση "handycrm"
   - Import → Επιλέξτε το database/handycrm.sql
   - Πατήστε "Go"
   
   Via Command Line:
   mysql -u root -p handycrm < database/handycrm.sql

3. Δημιουργήστε το config.php:
   
   - Αντιγράψτε το config/config.example.php
   - Μετονομάστε το σε config/config.php
   - Ενημερώστε τις ρυθμίσεις της βάσης:
     * DB_HOST: localhost
     * DB_NAME: handycrm
     * DB_USER: root (ή το δικό σας)
     * DB_PASS: το password σας

4. Ρυθμίστε δικαιώματα:
   
   chmod 755 uploads/
   chmod 644 config/config.php

5. Ανοίξτε το σύστημα:
   
   http://localhost/handycrm/

6. ΑΝΤΙΜΕΤΩΠΙΣΗ ΠΡΟΒΛΗΜΑΤΩΝ
══════════════════════════════════════════════════════════════════════════

❌ Πρόβλημα: "Σφάλμα σύνδεσης στη βάση δεδομένων"
   Λύση: Ελέγξτε ότι η MySQL τρέχει και τα credentials είναι σωστά

❌ Πρόβλημα: "Το αρχείο handycrm.sql δεν βρέθηκε"
   Λύση: Βεβαιωθείτε ότι το αρχείο database/handycrm.sql υπάρχει

❌ Πρόβλημα: "Permission denied" κατά τη δημιουργία config.php
   Λύση: Δώστε write permissions στο φάκελο config/
   chmod 755 config/

❌ Πρόβλημα: Λευκή σελίδα μετά την εγκατάσταση
   Λύση: Ενεργοποιήστε error reporting στο php.ini
   display_errors = On
