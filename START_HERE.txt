Γεια σου! 👋

Αυτό το folder περιέχει ΟΛΑ όσα χρειάζεσαι για να αναβαθμίσεις 
το HandyCRM στο ecowatt.gr/crm από την v1.3.6 στην v1.3.7.

Αυτή τη φορά είναι ένα ΟΛΟΚΛΗΡΩΜΕΝΟ release package με:
✅ Όλα τα αρχεία
✅ Database migrations
✅ Validation tools
✅ Step-by-step οδηγίες
✅ Troubleshooting guides

Δεν θα χρειαστεί να ψάχνεις για αρχεία ή να ανεβάζεις ένα-ένα!

═══════════════════════════════════════════════════════════════

📖 STEP 1: Διάβασε αυτά τα αρχεία με τη ΣΕΙΡΑ:

1️⃣ RELEASE_COMPLETE.txt (5 min)
   → Συνολική επισκόπηση του release
   → Τι διορθώνει, τι προσθέτει
   → Overview όλων των αλλαγών

2️⃣ RELEASE_1.3.7_README.md (10 min)
   → ΑΝΑΛΥΤΙΚΕΣ οδηγίες βήμα-βήμα
   → Troubleshooting tips
   → Πρέπει να το διαβάσεις ΟΛΟ!

3️⃣ DEPLOYMENT_CHECKLIST.txt (2 min)
   → Τύπωσέ το!
   → Τσεκάρε κάθε βήμα καθώς το κάνεις

═══════════════════════════════════════════════════════════════

🚀 STEP 2: Ακολούθησε αυτά τα βήματα:

1. Κάνε BACKUP (βάση + αρχεία) ⚠️ ΣΗΜΑΝΤΙΚΟ!
2. Τρέξε το RELEASE_1.3.7_MIGRATION.sql στο phpMyAdmin
3. Ανέβασε τα αρχεία (δες README για λίστα)
4. Edit το config.php ΧΕΙΡΟΚΙΝΗΤΑ (μόνο 2 συναρτήσεις)
5. Ανέβασε το validate_install.php και τρέξε το
6. Κάνε testing
7. Διάγραψε το validate_install.php

✅ Τι θα διορθωθεί:

1. ✅ Currency: "0,00 262145" → "0,00 €"
2. ✅ Labor editing: Δεν μηδενίζεται πλέον
3. ✅ Dates: MM/DD/YYYY → DD/MM/YYYY
4. ✅ PDF Reports: Δουλεύουν σωστά
5. ✅ User Roles: Όλοι οι 4 ρόλοι εμφανίζονται
6. ✅ Tasks: Προαιρετικά materials/labor
