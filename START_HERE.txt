🚀 3-STEP INSTALLATION
━━━━━━━━━━━━━━━━━━━━━━━━━━━━━

STEP 1: UPLOAD
   Extract this ZIP file
   Upload all files to your web server
   (FTP, cPanel File Manager, or any file upload method)

STEP 2: CREATE DATABASE
   Create a MySQL database in your hosting panel
   Write down: database name, username, password

STEP 3: VISIT YOUR URL
   Open your browser and go to your HandyCRM URL
   Example: https://yourdomain.com/handycrm/
   
   ⭐ The installation wizard will start AUTOMATICALLY!
   
   Just fill in:
   - Database details (from Step 2)
   - Admin username & password
   - Company information
   
   Click install and you're done! 🎉
